# DragonSky Co.,Ltd. - Static Website

A simplified static website for DragonSky Co.,Ltd., a mobile game developer specializing in warship and tank war games.

## Features

- **Clean and Modern Design**: Minimalist design with gradient backgrounds
- **Responsive Layout**: Works on desktop, tablet, and mobile devices
- **Interactive Slideshow**: Auto-playing hero section with manual controls
- **Video Integration**: Embedded YouTube video for game showcase
- **App Store Links**: Direct links to App Store and Google Play
- **Smooth Scrolling**: Enhanced navigation experience

## Structure

- `index.html` - Main HTML file with inline CSS and JavaScript
- `images/` - Directory containing all image assets
  - `logo.png` - Company logo
  - `game1.jpeg` - Game screenshot 1
  - `game2.jpeg` - Game screenshot 2
  - `game3.jpeg` - Game screenshot 3
  - `game4.jpeg` - Game screenshot 4
  - `app-store.png` - App Store download button
  - `google-play.png` - Google Play download button
- `README.md` - This file

## Sections

1. **Header**: Logo and navigation menu
2. **Hero Section**: Auto-playing slideshow with controls
3. **Games Section**: Video showcase and app store download links
4. **About Us Section**: Company information and game screenshots
5. **Privacy Policy Section**: Privacy information
6. **Footer**: Copyright information

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript (for slideshow functionality)
- No external dependencies

## Deployment

This is a static website that can be deployed to:

- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

### GitHub Pages Deployment

1. Push this repository to GitHub
2. Go to repository Settings > Pages
3. Select the branch (usually `main` or `master`)
4. Your site will be available at `https://[username].github.io/[repository-name]`

## Customization

### Colors

The color scheme uses CSS gradients. You can modify the colors in the `<style>` section:

- Hero slides: Modify `.slide` background gradients
- Section backgrounds: Modify `.about-content` background
- Accent color: Modify `#667eea` and `#764ba2` throughout

### Content

- Update company information in the About Us section
- Replace placeholder images with actual game screenshots
- Update app store links with your actual app URLs
- Modify the video embed URL in the Games section

### Images

All images are stored in the `images/` directory:
- Logo and game screenshots are already configured
- App store buttons are included
- All images use local paths (no external dependencies)
